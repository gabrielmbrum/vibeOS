# 📌 Guia de CMake

🔧 Instalação

```bash
sudo apt update
sudo apt install cmake
cmake --version
```

---

🏃‍➡️ Executando

1. entra no /build
2. cmake ..
3. make
4. rodar o exec (ex.: ./nome_do_executavel)

---

💻 Codando

Quando adicionar novo arquivo .c não se esquecer de alterar o "CMakeLists.txt"
